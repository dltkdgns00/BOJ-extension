# BOJ-Extension (Solid Lab Edition)

## 📢 공지사항 (Notice)
BOJ-Extension이 **v2.0 대규모 업데이트**를 맞이하여 소스 코드의 관리 정책이 변경되었습니다.
이에 따라 실제 소스 코드는 **Private Repository**로 전환되었음을 알려드립니다.

## 🎯 목적 (Purpose)
이 레포지토리는 사용자 여러분과의 소통을 위한 **공식 이슈 트래커(Issue Tracker)**로 운영됩니다.
BOJ-Extension 사용 중 발견된 **버그 제보**나 더 나은 도구를 위한 **기능 제안**은 [Issues](https://github.com/SolidLab-dev/BOJ-extension/issues) 탭을 이용해 주시기 바랍니다.
